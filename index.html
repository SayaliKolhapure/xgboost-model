About CancerGPT
A machine learning framework for cancer drug sensitivity prediction from gene expression data.
GDSC2 drug response147,490 labelled observations · 286 drugs · 969 cell lines
Gene expressionAffymetrix HG-U133A · 789 cell lines · 22,277 probes · top 2,000 by variance retained
Merged dataset621 cell lines · 95,994 drug-response observations · 31 TCGA cancer types
Model architectureXGBoost + SMOTE oversampling + SelectKBest (ANOVA-F, k=50) · per-drug binary classifiers
Evaluation5-fold stratified cross-validation · ROC-AUC and Average Precision
PerformanceMean AUC 0.798 · 33/41 models ≥ 0.75 · Best: Nutlin-3a AUC = 0.951
IC50 binarisationPer-drug z-score · sensitive = z ≤ −0.5 · resistant = z ≥ +0.5 · middle band excluded
Best modelNutlin-3a (MDM2 inhibitor) · AUC 0.951 · AP 0.925 · serialised as cancergpt_best_model.pkl
DisclaimerResearch tool only · not validated for clinical use · all outputs require oncologist review
